## Radar Generator

https://radar-generator.netlify.app/

Make Statsbomb like radars for (almost) any football players in the big 5 Euro leagues + MLS + FA WSL

Data scraped in real time from FBref

Tech Stack- React, D3
Backend hosted in AWS Lambda
